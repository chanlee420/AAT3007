Everything
- Very Fast File Explorer

Everything은 기존 Windows의 파일 탐색기와 비슷한 기능을 하는 프로그램입니다.
컴퓨터가 켜질때 저장소 내부의 모든 데이터를 색인으로 만들어 가지고 있기 때문에
검색과 동시에 모든 데이터를 확인하는 Windows 파일 탐색기에 비해서 빠른 성능을 가지고 있습니다.
프로그램 검색, 파일 검색, 폴더 검색 등 다양한 조건을 이용해서 검색을 할 수도 있으며,
정규식 등을 이용해 원하는 파일을 확실하게 찾을 수도 있습니다.
