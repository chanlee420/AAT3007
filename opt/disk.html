디스크 정리
- Clear out unused data from your storage

PC를 사용하다보면 저장소에는 어쩔 수 없이 사용하지 않는 데이터나, 임시 데이터들이 저장됩니다.
대표적으로 임시파일, 히스토리, 휴지통 내부의 삭제 파일 등이 있겠죠.
이런 파일들은 당장 용량을 많이 차지하지는 않지만 시간이 지나면서 계속해서 쌓이게 되고,
이후에 불필요할 정도로 쌓이게 되면 PC의 성능에 영향을 줄 정도가 됩니다.
이런 일을 방지하기 위해서 Windows 자체적으로 지원하는 기능인, 디스크 정리를 사용할 수 있습니다.
위의 이미지처럼 윈도우 검색기능에서 '디스크 정리'를 선택해 사용할 수 있으며,
원하는 파일만 정리할 수 있습니다.
자주 정리하는 경우 위 사진처럼 몇십 MB가 나오지만, 실제로 본 PC중에 몇GB 단위로 쌓여있는 경우도 있었습니다.
정리 가능한 파일 목록(클릭해서 펼쳐보기)

Microsoft Defender 바이러스 백신
Windows 업그레이드 로그 파일 다운로드한 프로그램 파일
임시 인터넷 파일
시스템 오류 minidump 파일
Windows 오류 보고서 및 피드백 진단
DirectX 셰이더 캐시
전송 최적화 파일
장치 드라이버 패키지
언어 리소스 파일
휴지통
임시 파일
미리 보기 사진
사용자 파일 히스토리

참고하실 점은, 디스크 정리 기능은 Windows 10과 11에서만 지원합니다.
