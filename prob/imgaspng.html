Save Image as PNG
- Download PNG from internet

Save Image as PNG는 이름 그대로, 이미지를 PNG로 저장할 수 있는 Chrome Extension입니다.
대부분의 경우 이미지를 우클릭해서 다운로드 받으려고 하면 webp 확장자로 다운로드 되며,
해당 확장자는 편집에 불편함이 있기 때문에 한번 더 확장자 변환이 필요합니다.
하지만 이 Extension을 사용하면 우클릭을 이용해서 PNG로 바로 다운로드가 가능합니다.


왼쪽은 기존의 다운로드 방법으로, webp 확장자로 다운로드 되며
오른쪽은 Save Image as PNG를 이용한 다운로드 방법으로, PNG 확장자로 다운로드 됩니다.
참고하실 점은, Chrome Extension이기 때문에 Chrome 브라우저가 아닌 타 웹브라우저는 사용이 불가능합니다.
